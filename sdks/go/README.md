# Go API client for Asguard

Complete Asguard platform SDK - Fraud Detection + Face Verification.
Two independent microservices, one unified client.

## Overview

This API client was generated by the [OpenAPI Generator](https://openapi-generator.tech) project. By using the [OpenAPI-spec](https://www.openapis.org/) from a remote server, you can easily generate a robust typed Go client.

- API version: 1.0.0
- SDK Package: `github.com/org-cyber/asguard/sdks/go`

## Installation

Install via `go get`:

```sh
go get github.com/org-cyber/asguard/sdks/go
```

## Getting Started

Using the Asguard Go SDK requires setting up two API clients if you are utilizing both services, as they operate independently on different ports (Fraud on `8081` and Face on `8082`).

Here is an extensive example covering the complete lifecycle: Health checks, Fraud transaction analysis, and Face verification + comparison.

### Example: Complete Workflow

```go
package main

import (
	"context"
	"encoding/base64"
	"fmt"
	"os"

	asguard "github.com/org-cyber/asguard/sdks/go"
)

func main() {
	ctx := context.Background()

	// ============================================
	// 1. Initializing the Fraud Detection Client
	// ============================================
	fraudConfig := asguard.NewConfiguration()
	fraudConfig.Host = "localhost:8081"
	fraudConfig.Scheme = "http"
	// Set the required API Key header for the Fraud service
	fraudConfig.AddDefaultHeader("X-API-Key", "devsecret")

	fraudClient := asguard.NewAPIClient(fraudConfig)

	// ============================================
	// 2. Initializing the Face Verification Client
	// ============================================
	faceConfig := asguard.NewConfiguration()
	faceConfig.Host = "localhost:8082"
	faceConfig.Scheme = "http"
	// Set the required Bearer token for the Face service
	faceConfig.AddDefaultHeader("Authorization", "Bearer dev-key-123")

	faceClient := asguard.NewAPIClient(faceConfig)

	// ============================================
	// 3. Executing a Fraud Check
	// ============================================
	fmt.Println("--> Running Fraud Check...")

	// Struct pointers map gracefully via SDK utility methods provided.
	deviceId := "device-abc"
	ipAddress := "192.168.1.100"

	req := asguard.FraudCheckRequest{
		TransactionId: "sdk-go-001",
		Amount:        250.00,
		Currency:      "USD",
		UserId:        "user-123",
		DeviceId:      &deviceId,
		IpAddress:     &ipAddress,
	}

	result, httpResp, err := fraudClient.FraudDetectionAPI.CheckFraud(ctx).FraudCheckRequest(req).Execute()
	if err != nil {
		fmt.Printf("Fraud API ERROR: %v\n", err)
	} else {
		fmt.Printf("Fraud API Status: %d\n", httpResp.StatusCode)
		fmt.Printf("Risk Level: %s (Score: %.1f)\n", result.RiskLevel, result.RiskScore)
		fmt.Printf("AI Opinion: %v\n\n", result.AiRecommendation)
	}

	// ============================================
	// 4. Executing Face Analysis & Comparison
	// ============================================
	fmt.Println("--> Running Face Verification...")

	// Replace with your actual base64 image data
	dataURI := "data:image/jpeg;base64,/9j/4AAQSkZJR..."

	analyzeReq := asguard.AnalyzeFaceRequest{
		Image:         dataURI,
		QualityChecks: asguard.PtrBool(true), // Enable quality assessments
	}

	analyzeResp, _, err := faceClient.FaceVerificationAPI.AnalyzeFace(ctx).AnalyzeFaceRequest(analyzeReq).Execute()

	if err != nil || !analyzeResp.Success {
		fmt.Printf("Analyze Failed: %v\n", err)
		return
	}

	fmt.Printf("Face Extracted! Embedding Length: %d, Quality: %.2f\n",
               len(analyzeResp.Embedding), *analyzeResp.QualityScore)

	// Compare the extracted embedding against another image
	compareReq := asguard.CompareFacesRequest{
		ProbeImage:         dataURI,
		ReferenceEmbedding: analyzeResp.Embedding,
		Threshold:          asguard.PtrFloat32(0.6), // Default allowed threshold
	}

	compareResp, _, err := faceClient.FaceVerificationAPI.CompareFaces(ctx).CompareFacesRequest(compareReq).Execute()

	if err == nil && compareResp.Success {
		fmt.Printf("Match: %v (Confidence: %.2f%%)\n", compareResp.Match, compareResp.Confidence * 100)
	}
}
```

## Advanced Configuration

The SDK permits overwriting configurations contextually. You can override contexts per API request to swap keys or host locations without overriding the defaults using Go's standard `context.WithValue`.

### Switching API Keys via Context

```go
authCtx := context.WithValue(
    context.Background(),
    asguard.ContextAPIKeys,
    map[string]asguard.APIKey{
        "ApiKeyAuth": {Key: "DIFFERENT_API_KEY"},
    },
)

r, hr, err := client.FraudDetectionAPI.CheckFraud(authCtx).FraudCheckRequest(req).Execute()
```
